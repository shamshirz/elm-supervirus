## Do stuff

* Just once
  * `npm install -g elm elm-format yarn`
* Whenever you change a dependency
  * `npm run prep`
* When you're working
  * `npm run start`
* Tests (from Root)
  * `elm-test`
* Build a prod version (minified)
  * `npm run deploy`
